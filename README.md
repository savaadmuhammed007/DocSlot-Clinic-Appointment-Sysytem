# 🏥 DocSlot - Django Clinic Appointment System

A simple web-based clinic management system built with **Django**.  
This project helps clinics manage patients, doctors, and appointment bookings efficiently.

---

## 🚀 Features

- Patient registration and login
- Book appointments with doctors
- View appointment history
- Doctor dashboard to view appointments
- Add consultation notes (diagnosis, prescription)
- Admin can manage doctors, departments, and schedules
- Prevents double booking of appointment slots

---

## 🛠️ Technologies Used

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
